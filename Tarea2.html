¿Cuáles són?:



  • Lenguaje de programación de bajo nivel:
    Se denominan lenguajes de maquina porque crea un vinculo entre el hardware y el software, entre estos:

    • Lenguaje de maquina:

      Es el código binario donde solo se utiliza 0 y 1.

    • Lenguaje Ensamblador

      Es el lenguaje más próximo que sería usado por el ser humano, este lenguaje es almacenado en texto y transforma este lenguaje a el de máquina.

  • Lenguaje de programación de alto nivel:
    Fácilita el trabajo del programador, utiliza notaciones matemáticas como c=a+b y después se traspasa al lenguaje de máquina.

    • Java

      Es un lenguaje de programación que tiene muchos años y por lo mismo muchas personas saben utilizarla, representa una pieza fundamental en demasiadas compañias ya que se utiliza en front de web, aplicaciones,etc.

    • Python

      Es el lenguaje más popular actualmente por simplificar demasiado el código y haciendolo más fácil de comprenderlo, se pueden crear bots,un desarrollo web,etc.

    • C++

      Es un lenguaje de programación orientado a objetos y se puede crear datos complejos, relacionar datos, definir operaciones sobre datos complejos,etc.
