Lenguajes de programación

Tarea #03

¿Qué son?:



Un lenguaje de programación es decirle a tu computadora una serie de instrucciones para llevar a cabo una idea mediante un código.


  • Clasificación de los lenguajes de programación de alto nivel

    • Primera generación:

      Fueron los lenguajes de máquina y ensamblador.

    • Segunda generación

      Entran los lenguajes imperativos como FORTRAN o COBOL.

    • Tercera generación:

      Siguen siendo lenguajes imperativos pero mucho más desarrollados como PASCAL o MEDULA

    • Cuarta generación:

      Se utilizan para bases de datos, ejemplos de esto son SQL y NATURAL

    • Quinta generación:

      Son creados para la inteligencia artificial como lo es PROLOG o LISP


  • Ejemplos de lenguajes:

    1. Interpretados:

      • Ruby
      • Python
      • Javascript

    2. Compilados

      • Rust
      • C
      • Java

    3. Orientado a objetos:

      • ADA
      • C++
      • Eiffel

    4. Funcionales:

      • Scala
      • ML
      • Lisp
