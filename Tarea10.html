Introducción



Tiene ese nombre en honor a Kurt Gödel y es un lenguaje fuertemente tipado con un sistema de tipos basados en many sorted logic con polimorfismo parametizado.

Símbolos



Los números se construyen con referencia a los símbolos

  • ¬es 1
  • ∀es 2
  • → es 3
  • ∨ es 4
  • ∧es 5


Sintaxis



  • Admite conjuntos finitos

  • Permite enteros y racionales de precisión arbitraria y números de punto flotante

  • Se basa en módulos

Ejemplo





MODULE      MCD.
IMPORT      Integers.
 
PREDICATE   Mcd : Integer * Integer * Integer.
Mcd(i,j,d) <- 
           DivisorComun(i,j,d) &
           ~ SOME [e] (DivisorComun(i,j,e) & e > d).
 
PREDICATE   DivisorComun : Integer * Integer * Integer.
DivisorComun(i,j,d) <-
           IF (i = 0 \/ j = 0)
           THEN
             d = Max(Abs(i),Abs(j))
           ELSE
             1 =< d =< Min(Abs(i),Abs(j)) &
             i Mod d = 0 &
             j Mod d = 0.
