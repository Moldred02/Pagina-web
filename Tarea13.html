Introducción



Es un lenguaje de programación que se adapta al desarrollo web donde es de código abierto y se puede comercializar muy fácil, su código se ejecuta en el servidor donde al leer los comandos puede activar todos los elementos funcionales y la interfaz visual del sitio web

Símbolos



  • + es suma
  • - es resta
  • * es multiplicación
  • .= son operadores de asignación
  • ?? son operadores de comparación
  • ??= son operadores de asignación
  • / es División
  • % es Modulo
  • = es Asignación


Sintaxis



  • Las variables son declaradas con el signo $

  • Los comentarios pueden ser // o #

  • Para delimitar el codigo se puede poner:
    • < ?""? >
    • < %""% >
